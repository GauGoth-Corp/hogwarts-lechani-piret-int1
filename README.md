# Hogwarts Game - v. 0.1
Hogwarts Game is a parodic text-based adventure game set in the _Harry Potter_ universe. This project is developed as part of a Python programming project at our school **Efrei Paris**.

_Developed by Aomm-lgtm & GauGoth Corp._ (**Lisa Lechani** & **Gautier Piret**)


## How to play

### Installation
_If you encounter any issue during installation, do not hesitate to [contact us](http://gaugoth.corp.free.fr/en/credits/contact/?subject=Hogwarts%20Game%20v.%200.1%20installation%20issue)!_

1. If not already installed, download and install Python 3.11 or higher from the official website: [python.org](https://www.python.org/downloads/).
2. Download the Hogwarts Game source code from this link: [[hogwarts-lechani-piret-int1-main.zip](https://github.com/GauGoth-Corp/hogwarts-lechani-piret-int1/archive/refs/heads/main.zip)].
3. Extract the downloaded ZIP file to the desired location on you computer.
4. Open the folder `hogwarts-lechani-piret-int1-main`, then navigate to the `hogwarts` folder.
5. Open a terminal in this folder (on Windows, you can do this by right-clicking then selecting "Open in Terminal").
6. Type the following command to run the game:
   ```
   python main.py
   ```
   If not working, try using `py` or `python3` instead of `python`.
7. Have fun!


### Features & Gameplay
- Parodic text-based adventure game
- The player will follow the adventures of Harry Potter during his first year at Hogwarts and make its own choices influencing the story
- The player can die for certain bad choices!
- The game is divided into 4 chapters. The last one is inspired from the 4th book: _Harry Potter and the Goblet of Fire_ and the dragon challenge of the Triwizard Tournament
- Simple RPG system with attributes (Courage, Intelligence, Loyalty, Ambition), inventory, money, spells, House, Health Points, Energy Points, etc.
- Home Menu with several options (Start Game, Load Game, Credits, Contact Us, Quit Game)
- Character creation (first name, last name, attributes values)
- **Notice: An easter egg is hidden somewhere!**
